# 0015. 새 포스트: 위치 추가 + 활동별 태그

- 상태: 채택
- 날짜: 2026-07-09

## 배경

포스트에 장소를 지정할 방법이 없었고(핀=현재 위치 고정), 태그가 활동과 무관한 고정 6개뿐이라 좁았다.

## 결정

- 위치 추가: 체크인과 같은 인스타식 장소 검색(pick-location + usePlacePicker 재사용). 고르면 posts.place_name 저장 + 핀 좌표를 그 장소로. 피드 카드에 📍 장소명 표시.
- 활동별 태그: post-tags.ts에서 활동(스키/백컨트리/하이킹/러닝/MTB 등)별 추천 태그 + 공통 태그를 합쳐 노출. 선택 태그는 활동을 바꿔도 유지.

## 결과

- 정보의 정확도(장소)와 태그 표현력이 크게 향상. 감수: place_name은 자유 지오코딩 결과라 표기 편차 존재.
