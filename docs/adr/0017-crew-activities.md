# 0017. 크루 활동 연결

- 상태: 채택
- 날짜: 2026-07-09

## 배경

크루(그룹)는 만들었지만, 체크인·포스트가 크루와 연결되지 않아 "크루의 활동"이 쌓이지 않았다. 소속감·재방문을 만들려면 활동을 크루에 귀속시켜야 한다.

## 결정

posts.crew_id로 포스트(및 체크인 공유 포스트)를 크루에 연결.
- 작성 시 "크루 활동으로" 선택기(내가 가입한 크루). 체크인 공유 포스트도 crew_id를 가짐.
- 크루 활동 포스트 등록 시 트리거가 크루원(작성자 제외)에게 send_expo_push 알림.
- crew_posts(cid) RPC(멤버 게이트, security definer)로 크루 홈에 활동 피드 노출 — 만료 무관 최근 50건(활동 기록 보존).
- 벤치마크: Strava 클럽 피드, 밴드 그룹 피드.

## 대안

- 크루 전용 별도 포스트 테이블: 중복·복잡 — posts에 crew_id 부여로 재사용.

## 결과

- 크루 단위로 활동이 모이고 알림으로 참여를 유도. 감수: 비공개(공유 off) 체크인은 포스트가 없어 크루에 안 올라감(의도된 동작).
