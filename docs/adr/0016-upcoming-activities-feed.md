# 0016. 피드 '다가오는 활동' 분리

- 상태: 채택
- 날짜: 2026-07-09

## 배경

예약(미래 시작) 체크인을 공유하면 '지금 주변에서' 피드에 섞여, "지금 정보"라는 정체성과 어긋나고 언제 시작인지도 불명확했다.

## 결정

예약 체크인 포스트(scheduled_start_at 미래 + '예정' 태그)를 피드 상단 **'다가오는 활동' 가로 카러셀**로 분리. 시작 시간 순 정렬 + D-day 표시. '지금' 피드와 홈 미리보기에서는 제외. 시작 시각이 지나면 자동으로 일반 피드('진행 중')로 편입.

- 만료 로직은 변경하지 않음(사용자가 이벤트-연동 만료는 원치 않음). nearby_posts v6이 연결 체크인의 scheduled_start_at만 추가로 반환.

## 대안

- 인라인 '예정' 배지만: 지금/미래 혼재 지속 — 기각.
- '예정' 필터 칩: 능동적으로 눌러야 발견 — 기각.

## 결과

- '지금'과 '예정'이 분리돼 피드가 깔끔하고 예약 활동 모집이 부각됨. 감수: 포스트 48h 만료를 안 바꿔, 48시간보다 먼 예약은 카러셀에서도 조기 하차(후속 여지).
