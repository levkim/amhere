# 0004. 지도: Mapbox

- 상태: 채택
- 날짜: 2026-07-06

## 배경

앱의 홈 화면이 지도다. 아웃도어(산악 지형) 표현과 향후 오프라인 지도가 중요하다. 단, Mapbox는 네이티브 모듈이라 Expo Go에서 동작하지 않고 EAS dev build가 필요하다.

## 결정

`@rnmapbox/maps` (Mapbox)를 사용한다. Outdoors 스타일로 지형을 표현하고, 웹에서는 `live-map.web.tsx` 폴백을 보여준다. 지도 도입 시점부터 dev build 기반 개발로 전환했다.

## 대안

- **OpenStreetMap 계열 (무료, Expo Go 호환)**: 진입은 쉽지만 아웃도어 지형·오프라인 지도가 약해 이 앱의 핵심 가치와 안 맞음
- **Google Maps**: 아웃도어 특화 스타일 부족, 오프라인 제약

## 결과

- AllTrails급 아웃도어 지도 표현력, v1 오프라인 지도 확장 경로 확보
- 감수: 무료 티어 초과 시 과금, 빌드에 비밀 다운로드 토큰(sk.) 필요 — eas.json은 git 제외
